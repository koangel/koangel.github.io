Boost自动编译程序 for Python

使用Python编写的BOOST程序

引子

编写此程序的缘由在于可以不用一直切换VS的环境，并且可以自动编译64位或32位程序，更加傻瓜的方式，后期我可能会写一个UI程序，这样点选即可。

ChangeLog

Python写的Boost自动编译程序
Version:1.1 beta

2016-07-16：

  • 根据自己的环境筛选出已安装环境，并列出 环境提供选择。
  • 支持X64位的自动参数编译,（请务必注意修改源码设置路径）。
  • 可以选择编译的参数，其他版本持续改进中
  • 自动编译自动安装到指定目录（需要自己改PY代码）
  • 有需要改进的请继续，自行修改，修改请保留作者版权。
  • 已测试过 WIN10下自动编译B2和编译VS2010版BOOST 1.61
  • 目前仅支持VS2010以及未来版本，仅支持BOOST 1.53以上版本

源码

用法

  • 安装Python 2.7版本
  • 首先下载脚本，并修改脚本内参数的BOOST路径,例如:
    boost_work_path = "E:/boost/boost_1_61_0"
  • 运行脚本设置选项
